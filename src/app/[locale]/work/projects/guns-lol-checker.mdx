## Guns.lol Otomatik Kullanıcı Adı Denetleyici

Guns.lol platformu üzerinde boşta olan ve alınabilir kullanıcı adlarını tespit etmek için geliştirdiğim bu araç sayesinde, manuel deneme yanılma sürecini tamamen otomatize ettim. 

**Projenin Öne Çıkan Özellikleri:**
- **Asenkron HTTP İstekleri:** Hızlı tarama yapabilmek için çoklu istek (multithreading/asynchronous) mimarisi kullanılmıştır.
- **Hata Yönetimi (Rate Limit Handling):** API veya web servislerine yapılan isteklerde karşılaşılan oran sınırlarına (rate limit) karşı otomatik bekleme ve yeniden deneme mekanizması içerir.
- **Performans:** Yüksek hızda çalışarak binlerce kombinasyonu saniyeler içerisinde test edebilir.
