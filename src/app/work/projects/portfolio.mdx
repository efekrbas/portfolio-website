## Kişisel Portfolyo Web Sitesi

Kişisel deneyimlerimi, projelerimi ve blog yazılarımı paylaştığım web sitemdir. Daha önceden Vanilla HTML/CSS/JS ile geliştirilmiş olup, güncel olarak Next.js ve MagicUI şablonu kullanılarak modern ve performanslı bir altyapıya taşınmıştır.

**Kullanılan Teknolojiler:**
- Next.js
- React
- Tailwind CSS
- MagicUI
